# 新中国特色单位换算器 🇨🇳

一个基于Web的中国特色社会计量单位转换工具，支持「钿」「爽」「坤」等网络流行计量单位的互转。

崩项目采用 GPL-3 许可协议，欢迎广大人民群众转发、修改及备份。


## 背景 📚

本项目收录近年来在中国互联网和社会经济领域产生的特色计量单位，这些单位反映了：
- 网络文化现象
- 社会收入认知
- 民生民意思想
- 流行文化符号
